# 5年生 算数プリント - 拡張版 (AI作成問題集)

## 概要

「現場の先生が本当に使いやすい」を目指した、小学5年生全単元対応の算数プリント集です。
テスト分析結果と**学習指導要領（平成29年告示）**に基づき、児童がつまずきやすいポイントを重点的にカバーします。
各単元につき5つの難易度別プリント（基礎・標準・応用・思考力・まとめ）と、それぞれの解答を用意しています。

---

## 開発コンセプト: "Clean & Practical"

- **実用性第一**: 授業や宿題ですぐに使える、余計な装飾のないシンプルなデザイン。
- **視認性重視**: すべてのプリントで **UDデジタル教科書体** を採用し、十分な余白を確保。
- **筆算重視**: 特に5年生の「小数のかけ算・わり算」単元では、計算過程を書き込みやすい広さを確保。
- **学習指導要領準拠**: 評価の3観点（知識・技能、思考・判断・表現、主体的に学習に取り組む態度）を意識した問題設計。

## 5年生でのつまずきポイント（テスト分析より）

### AI問題作成への示唆

1. **小数の仕組み**: 「0.1を何個集めた数か」という構成理解が重要
2. **体積の単位換算**: ㎥⇔L⇔㎤の関係が複雑で要練習
3. **複合図形**: 「分ける」「引く」の複数解法を問う
4. **合同条件**: 証明の土台として重要
5. **計算の工夫**: 分配法則・結合法則の活用問題
6. **エラー分析**: 「間違っているところを直す」形式が効果的
7. **割合**: 基準量と比較量の区別が5年生最大のつまずき
8. **単位量あたりの大きさ**: 「どちらをどちらで割るか」の判断（1あたり量）が課題
