# 3年生 算数プリント - 拡張版 (AI作成問題集)

## 概要

「現場の先生が本当に使いやすい」を目指した、小学3年生全単元対応の算数プリント集です。
従来版から大幅に拡張し、各単元につき5つの難易度別プリント（基礎・標準・応用・思考力・まとめ）と、それぞれの解答を用意しています。

## 開発コンセプト: "Clean & Practical"

- **実用性第一**: 授業や宿題ですぐに使える、余計な装飾のないシンプルなデザイン。
- **視認性重視**: すべてのプリントで **16pt** の大きな文字（UDデジタル教科書体）を採用。
- **1枚完結**: どんなに内容が多くても、A4用紙1枚（片面）に収まるよう緻密にレイアウト調整済み。
- **Web連携**: GitHub Pagesで公開し、「問題」と「解答」をセットでダウンロード可能。

## 作成日

- **プロジェクト開始**: 2026年1月
- **最終更新**: 2026年1月12日 (全単元 16pt化完了)

## プリント構成（5段階レベル）

各単元（全18単元）に以下の5種類のプリントがあります。

1.  **基礎**: 計算の基本や用語の確認など、導入に最適。
2.  **標準**: 教科書レベルの標準的な問題。
3.  **応用**: 文章題や少し複雑な計算を含む、力のつく問題。
4.  **思考力**: 知識を活用して考える、ハイレベルな問題。
5.  **まとめ**: 単元の学習内容を総復習するテスト形式。

※ すべてに赤字の解答PDFが付属しています。

## 仕様

- **用紙**: A4サイズ（210mm × 297mm）
- **フォント**: モリサワ UDデジタル教科書体 ProN (Medium/Bold)
- **文字サイズ**: **16pt** (行間調整により視認性を最大化)
- **マージン**: 上下左右 5mm〜10mm (内容量に応じて最適化)
- **形式**: PDF (LaTeXにより自動生成)

## 開発・運用フロー

### 1. LaTeXソースの編集・修正
各単元の `src` フォルダ内の `.tex` ファイルを編集します。

### 2. PDFの一括生成
以下のコマンドで、修正した単元（または全単元）のPDFを再生成できます。
```bash
python3 recompile_all.py
```
※ `recompile_all.py` 内の `start_unit`, `end_unit` で範囲指定可能。

### 3. Webサイトの更新
`index.html` は以下のスクリプトで生成します。新しい単元を追加した場合などに実行します。
```bash
python3 generate_index.py > index.html
```

### 4. チェック
`check_page_counts.py` を実行すると、2ページ以上になってしまっているPDF（オーバーフロー）を検出できます。
```bash
python3 check_page_counts.py
```
