# 3年生 算数プリント - プロトタイプ

## 概要

3年生の全単元（1学期〜3学期）をカバーする算数プリントです。
テスト分析結果に基づき、各単元の重要ポイントとつまずきやすいポイントを考慮して作成しました。

## 作成日

- **初版作成**: 2026年1月10日
- **最終更新**: 2026年1月11日

## プリントの特徴

- **問題数**: 各プリント2〜3問（十分な書き込みスペースを確保）
- **ヒント**: すべての問題に学習支援のヒントを配置
- **フォント**: モリサワ UDデジタル教科書体（Medium/Bold）
- **サイズ**: A4サイズ（210mm × 297mm）
- **印刷**: グレースケール印刷に最適化

## プリント一覧（全18単元）

### 1学期（7単元）
1. **九九の表とかけ算** - 10の段、0のかけ算、九九のきまり、逆算
2. **わり算** - 基本的なわり算、あまりのあるわり算、逆算
3. **たし算とひき算の筆算** - 3桁の筆算、文章題、逆算
4. **時こくと時間** - 時間の単位換算、時刻と時間の計算
5. **一万をこえる数** - 大きな数の読み書き、位の理解、大小比較
6. **表とグラフ** - 表の読み取り、棒グラフの作成と読み取り
7. **たし算とひき算（工夫した計算）** - 暗算、計算の工夫

### 2学期（9単元）
8. **長さ** - km、m、cmの単位換算と計算
9. **重さ** - kg、gの単位換算と計算
10. **円と球** - 円の中心、半径、直径、球の性質
11. **小数** - 小数の表し方、小数の加減
12. **何倍でしょう** - 倍の概念、倍を求める計算
13. **分数** - 分数の表し方、同分母分数の加減
14. **□を使った式** - 未知数を含む式、逆算
15. **式と計算** - 四則混合計算、計算の順序、括弧の使用

### 3学期（3単元）
16. **三角形** - 正三角形、二等辺三角形の性質と作図
17. **総復習** - 全単元の総合問題
18. **かけ算とわり算** - かけ算の筆算、あまりのあるわり算の応用（新規追加）

## 使用方法

### PDFの印刷
1. `PDF/` フォルダ内のPDFファイルを開く
2. 印刷設定で「実際のサイズ」を選択
3. グレースケール印刷を推奨

### LaTeXファイルの編集・再生成
1. `src/` フォルダ内の.texファイルを編集
2. LuaLaTeXでコンパイル:
   ```bash
   cd src
   lualatex ファイル名.tex
   mv ファイル名.pdf ../PDF/
   ```

## 設計方針

### 1. 書き込みスペースの確保
- 計算問題: 15〜20mm
- 文章題: 20〜25mm
- 図形問題: 適宜調整

### 2. ヒントの配置
- 黄色の背景ボックス（tipbox）
- 各問題の下に配置
- つまずきポイントを重点的に説明

### 3. テスト分析の反映
テスト分析結果から以下を重点的に扱う:
- **単位換算** - km/m/cm、kg/g、時間の単位
- **計算の順序** - 四則混合、括弧の使用
- **図形の性質** - 正三角形、二等辺三角形、円
- **文章題** - 実生活との関連

### 4. UD（ユニバーサルデザイン）
- UDデジタル教科書体の使用
- 十分な行間・余白
- グレースケール印刷対応
- シンプルで読みやすいレイアウト
