# 2年生 算数プリント - 拡張版 (AI作成問題集)

## 概要

「現場の先生が本当に使いやすい」を目指した、小学2年生全単元対応の算数プリント集です。
低学年の子どもたちが楽しく取り組めるよう、**20ptの大きな文字**と**視覚的なアイコン（◯△□など）**を積極的に使用しています。

## 開発コンセプト: "大きく・わかりやすく・たのしく"

- **大きな文字**: すべてのプリントで **20pt** のフォントを採用（3〜6年生版は16pt）
- **ひらがな中心**: 2年生の既習漢字のみ使用、ふりがなを適切に付与
- **視覚的アイコン**: ◯△□★などの記号、🍎🐱などの絵文字を活用
- **1枚完結**: A4用紙1枚（片面）に収まるレイアウト

1. **基礎** ⭐: 導入・基本問題
2. **標準** ⭐⭐: 教科書レベル

## 注意事項

### 表とグラフ問題について
- 数値の整合性を必ず確認すること
- グラフの◯の数と表の数値が一致していること
- 子どもは細かいところによく気づくので、矛盾があると信頼を失う

### 低学年向け配慮
- 問題文は短く、ひらがな中心で
- 視覚的なアイコン（◯△□★🍎🐱など）を積極的に使用
- 余白を多めにとり、書き込みやすく
