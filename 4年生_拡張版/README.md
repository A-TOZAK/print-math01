# 4年生 算数プリント - 拡張版 (AI作成問題集)

## 概要

「現場の先生が本当に使いやすい」を目指した、小学4年生全単元対応の算数プリント集です。
テスト分析結果と**学習指導要領（平成29年告示）**に基づき、児童がつまずきやすいポイントを重点的にカバーします。
各単元につき5つの難易度別プリント（基礎・標準・応用・思考力・まとめ）と、それぞれの解答を用意しています。

---

## 開発コンセプト: "Clean & Practical"

- **実用性第一**: 授業や宿題ですぐに使える、余計な装飾のないシンプルなデザイン。
- **視認性重視**: すべてのプリントで **UDデジタル教科書体** を採用し、十分な余白を確保。
- **筆算重視**: 特に4年生の「筆算」単元では、計算過程を書き込みやすい広さを確保（2ページ構成を許容）。
- **学習指導要領準拠**: 評価の3観点（知識・技能、思考・判断・表現、主体的に学習に取り組む態度）を意識した問題設計。
